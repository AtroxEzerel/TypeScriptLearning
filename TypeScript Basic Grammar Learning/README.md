## TypeScript  Basic Grammer Learning

### 一.TypeScript简介

#### 1.TypeScript是什么

- 以JavaScript为基础构建的语言
- 一个JavaScript的超集
- 可以在任何支持JavaScript的平台执行
- TypeScript扩展了JavaScript 并添加了类型
- TS不能被JS解析器直接执行

TS===编译===>JS

#### 2.TypeScript增加了什么

- 类型
- 支持ES的新特性
- 添加ES不具备的新特性
- 丰富的配置选项
